A história da Internet

Iniciou para fins de defesa e hoje é parte essencial de nossas vidas.

Voltar

A história da Internet começa na década de 1960, quando o Departamento de Defesa dos Estados Unidos criou a ARPANET (Advanced Research Projects Agency Network), uma rede de computadores que permitia a troca de informações entre instituições de pesquisa.
Na década de 1970, a ARPANET cresceu e evoluiu para incluir outras redes, formando a base da Internet. O primeiro e-mail foi enviado em 1971 e o protocolo TCP/IP (Transmission Control Protocol/Internet Protocol) foi desenvolvido para permitir que diferentes redes se comunicassem.
Nos anos 1980, a Internet se expandiu para incluir universidades e organizações de pesquisa em todo o mundo, e a World Wide Web foi criada em 1989 por Tim Berners-Lee, um cientista da computação britânico.
A web permitia a criação de páginas interconectadas por links, tornando mais fácil a navegação e o acesso à informação.
Nos anos 1990, a Internet se tornou disponível para o público em geral e começou a mudar a maneira como as pessoas se comunicavam, trabalhavam e consumiam informações.
A criação de motores de busca, como o Google, permitiu que os usuários encontrassem informações com facilidade, e o surgimento de sites de comércio eletrônico revolucionou a maneira como as pessoas faziam compras.
Hoje, a Internet é uma parte essencial de nossas vidas, conectando bilhões de pessoas em todo o mundo e permitindo a troca de informações, ideias, conhecimentos e cultura.
A evolução contínua da Internet continua a moldar a maneira como vivemos e interagimos uns com os outros.

A linguagem HTML

A organização e estrutração da web moderna.

Voltar

O HTML (Hypertext Markup Language) é a linguagem de marcação padrão usada para criar páginas web.
Ele foi desenvolvido na década de 1990 por Tim Berners-Lee, um físico e cientista da computação britânico que trabalhava no CERN (Organização Europeia para a Pesquisa Nuclear).
Berners-Lee estava trabalhando em um projeto chamado World Wide Web (WWW), que tinha como objetivo permitir que cientistas de todo o mundo compartilhassem informações e documentos de pesquisa.
Para tornar isso possível, ele precisava de uma maneira de organizar e estruturar o conteúdo que seria publicado na web. Assim, o HTML foi criado. A primeira versão do HTML foi lançada em 1991 e tinha recursos básicos, como a capacidade de criar links entre documentos e a habilidade de exibir imagens.
Ao longo dos anos, novas versões do HTML foram sendo lançadas, adicionando recursos e funcionalidades mais avançadas, como suporte para formulários, vídeos e animações.
Hoje, o HTML é uma parte essencial da web e é usado em conjunto com outras linguagens de programação, como CSS (Cascading Style Sheets) e JavaScript, para criar sites e aplicativos web modernos e interativos.
A evolução contínua do HTML continua a moldar a maneira como interagimos e consumimos informações na web.

Client Side x Server Side

Entenda o funcionamento de um site ou app.

Voltar

O desenvolvimento de sites e aplicativos web envolve dois aspectos principais: o lado do cliente (client side) e o lado do servidor (server side).
O lado do cliente refere-se a tudo o que acontece no navegador do usuário. Isso inclui o processamento de HTML, CSS e JavaScript, bem como qualquer interação com o usuário, como cliques em botões ou preenchimento de formulários.
Todas essas atividades são executadas no dispositivo do usuário, como computador, tablet ou smartphone. As linguagens de programação usadas para desenvolver o lado do cliente incluem HTML, CSS e JavaScript. O HTML define a estrutura do conteúdo da página, o CSS define sua aparência visual e o JavaScript adiciona interatividade e dinamismo.
Já o lado do servidor refere-se ao processamento de informações e dados que são enviados e recebidos pelo servidor.
Isso inclui a manipulação de dados de usuários, a execução de consultas em bancos de dados e a geração dinâmica de conteúdo baseado em informações específicas do usuário. As linguagens de programação usadas para desenvolver o lado do servidor incluem PHP, Ruby, Python, Java, entre outras.
Essas linguagens são usadas para criar aplicativos web dinâmicos que podem se comunicar com bancos de dados, processar dados de formulários e gerar conteúdo personalizado para cada usuário. Geralmente, um aplicativo web é dividido em duas partes, o front-end (client side) e o back-end (server side), cada um sendo responsável por diferentes aspectos do funcionamento do aplicativo.
A comunicação entre as duas partes ocorre por meio de APIs (interfaces de programação de aplicativos) e protocolos de comunicação, como o HTTP. O desenvolvimento de aplicativos web envolve conhecimentos em ambas as áreas (client side e server side), bem como em outras áreas relacionadas, como design de interface do usuário, experiência do usuário e segurança.
É importante que os desenvolvedores tenham uma compreensão completa do processo de desenvolvimento de aplicativos web para criar aplicativos eficientes, seguros e eficazes.

As principais tags HTML

Voltar

O HTML utiliza tags para estruturar e formatar o conteúdo. Algumas das tags mais comuns incluem:

  • <h1> até <h6> - utilizadas para definir títulos e subtítulos;
  • <p> - utilizada para definir parágrafos de texto;
  • <mark> - utilizada para destacar trechos de texto;
  • <small> - utilizada para definir um texto menor e com menos destaque;
  • <i> - utilizada para definir texto em itálico;
  • <u> - utilizada para definir texto sublinhado;
  • <ol> - utilizada para criar uma lista ordenada;
  • <ul> - utilizada para criar uma lista não ordenada;
  • <li> - utilizada para criar um item de lista;
  • <a> - utilizada para criar links para outras páginas ou recursos na web;
  • <hr> - utilizada para criar uma linha horizontal;
  • <sub> - utilizada para definir texto subscrito;
  • <sup> - utilizada para definir texto sobrescrito;
  • <blockquote> - utilizada para definir citações em bloco.

Curiosidades

Algumas informações curiosas.

Voltar

O primeiro e-mail foi enviado em 1971 por Ray Tomlinson. Ele escolheu o símbolo "@" para separar o nome do usuário do endereço de destino.

O primeiro site da história foi publicado em 1991 por Tim Berners-Lee, criador da World Wide Web. Ele era um site simples com informações sobre o projeto da WWW.

A primeira mensagem instantânea foi enviada em 1989 por um programador chamado Mirabilis. A empresa posteriormente criou o ICQ, um dos primeiros programas de chat populares.

O primeiro emoji foi criado em 1999 pelo designer japonês Shigetaka Kurita. Ele criou um conjunto de 176 ícones para serem usados em celulares da época.

A primeira webcam foi inventada em 1991 por Quentin Stafford-Fraser e Paul Jardetzky. Eles usaram a webcam para monitorar uma cafeteira no laboratório de informática da Universidade de Cambridge.

O primeiro nome registrado na história da internet foi Symbolics.com. O registro ocorreu em 15 de março de 1985.

A palavra "Google" é uma variação da palavra "googol", que se refere ao número 1 seguido de 100 zeros. O nome foi escolhido para representar a missão da empresa de organizar a enorme quantidade de informações disponíveis na web.

Em média, cerca de 300 bilhões de e-mails são enviados todos os dias em todo o mundo.

A maioria das pessoas passa mais tempo navegando na internet em seus telefones celulares do que em computadores desktop.

A internet não é uma entidade física única, mas sim uma rede global de computadores interconectados. Estima-se que existam mais de 4,5 bilhões de usuários de internet em todo o mundo
